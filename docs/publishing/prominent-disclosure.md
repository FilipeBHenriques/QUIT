# Prominent Disclosure Draft

Use this as the basis for an in-app disclosure shown before the user enables Accessibility Service and other special access permissions.

## Suggested Disclosure Text

QUIT uses Accessibility Service, Usage Access, and Display Over Other Apps to block the apps and websites that you choose.

These permissions are used only for QUIT's core blocking features:

- Accessibility Service helps detect supported browser website visits and app transitions needed for blocking
- Usage Access helps detect foreground app usage and support time tracking
- Display Over Other Apps lets QUIT show the blocking screen above blocked content

QUIT stores your blocked apps, blocked websites, and app state locally on your device. QUIT also uses Google AdMob for advertising.

By continuing, you choose to enable these permissions for QUIT's blocking features.
