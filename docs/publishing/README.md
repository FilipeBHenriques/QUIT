# Publishing Pack

This folder contains draft materials for Google Play submission.

## Files

- `google-play-listing.md`
  - app title
  - short description
  - full description
- `privacy-policy.md`
  - publish this at a public web URL before Play submission
- `google-play-review-notes.md`
  - use this for reviewer notes and accessibility declaration context
- `google-play-data-safety.md`
  - use this as a worksheet when filling Play Console Data Safety
- `prominent-disclosure.md`
  - use this as the basis for an in-app permission disclosure
- `google-play-assets.md`
  - screenshot and feature graphic checklist

## Important

These documents improve your review readiness, but they do not guarantee approval.

Google also reviews:

- actual app behavior
- the permissions flow
- AdMob and Data Safety answers
- accessibility justification
- the public privacy policy URL
