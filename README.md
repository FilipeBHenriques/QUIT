# QUIT

A simple app to block and unblock unwanted Android apps.

## Features

- Block distracting apps
- Overlay lock screen when blocked apps are opened
- Simple unblock process

## Getting Started

1. Clone the repo
2. Run `flutter pub get`
3. Run the app on your Android device

## Permissions

- Usage Access: Needed to detect running apps.
- Overlay: Needed to display lock screen over blocked apps.
